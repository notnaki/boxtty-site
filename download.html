Build boxtty

No notarized binary yet — boxtty builds from source in two commands. You'll need Xcode and XcodeGen (brew install xcodegen).

git clone https://github.com/notnaki/boxtty
cd boxtty/boxtty-mac
./build.sh
open dist/Boxtty.app
